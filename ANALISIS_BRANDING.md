# EVENTUS - Análisis de Branding

## PALETA DE COLORES

### Colores Primarios

| Color | Código | Uso | Propósito |
|-------|--------|-----|-----------|
| **Azul** | #0B334C | Fondos, headers, componentes principales | Transmite confianza, profesionalismo, control |
| **Verde menta** | #BCEE95 | Acentos, botones, isotipo | Energía, éxito, movimiento, verificación |
| **Verde menta claro** | #DEF3E0 | Fondos suaves, estados hover | Soft, accesible, complemento del verde principal |

### Colores Secundarios

| Color | Código | Uso |
|-------|--------|-----|
| **Blanco** | #FFFFFF | Texto principal, fondos limpios |
| **Negro** | #1D1D1D | Texto oscuro, componentes |

## TIPOGRAFÍA

### Font: Mont

- **Categoría:** Sans-serif moderna
- **Peso disponibles:** Regular, Bold, Semi-bold (por lo menos)
- **Uso:** 
  - Headings: Bold/Semi-bold
  - Body text: Regular
  - Botones: Semi-bold

**Características:**
- Legible y moderna
- Profesional sin ser seria
- Funciona bien en pantallas pequeñas y grandes
- Espaciado generoso

## IDENTIDAD VISUAL

### Isotipo

- Forma: Checkmark (marca de verificación)
- Color: Verde menta (#BCE95)
- Estilo: Redondeado, dinámico, seguro
- Significado: Verificación, éxito, control, aprobación
- Adaptabilidad: Funciona solo o con el logotipo

### Logotipo

- Composición: Checkmark + "eventus" (texto)
- Alineación: Horizontal
- Color del texto: Blanco sobre fondos oscuros
- Uso: Header principal, branding general

## ESTILO Y ESTÉTICA

### Características visuales

1. **Bordes redondeados** - Aplicados a:
   - Cards
   - Botones
   - Modales
   - Componentes

2. **Fondos sutiles** - Patrón de grid sutil visible en algunos contextos

3. **Sombras suaves** - Elevan componentes sin ser agresivas

4. **Espaciado generoso** - Mucho aire blanco alrededor de elementos

5. **Componentes humanizados** - Uso de fotografías reales de personas

### Atmósfera general

- Sobrio y premium
- Profesional pero accesible
- Moderno y contemporáneo
- Transmite control y organización
- Confiable y eficiente

## COMPOSICIÓN Y LAYOUT

### Patrón de colores típico

1. Fondo: Azul oscuro (#0B3344)
2. Texto principal: Blanco
3. Acentos: Verde menta (#BCE95)
4. Fondos secundarios: Verde menta claro (#DEF3E0)

### Proporciones de color recomendadas

- **Azul oscuro**: 60-70% (fondos, espacios principales)
- **Blanco**: 20-25% (texto, espacios)
- **Verde menta**: 5-10% (acentos, CTA)
- **Verde menta claro**: 5-10% (fondos suaves, estados)

## COMPONENTES OBSERVADOS

### Botones
- Fondo: Generalmente verde menta o azul oscuro
- Texto: Blanco o contraste alto
- Bordes: Redondeados (aprox. 8-12px)
- Estado hover: Verde menta claro o más oscuro

### Cards
- Fondo: Generalmente combinación de verde menta claro + blanco
- Bordes: Redondeados
- Sombra: Sutil
- Contenido: Imágenes + texto

### Headers/Modales
- Fondo: Azul oscuro
- Texto: Blanco
- Acentos: Verde menta

### Estados
- Verificado/Exitoso: Verde menta
- Error: Rojo (observado en badge de notificación)
- Deshabilitado: Gris o verde más claro

## APLICACIONES OBSERVADAS

1. **En Mobile** - Notificación con badge rojo
2. **En Modal** - Header azul + contenido blanco/verde
3. **En Desktop** - Dashboard con fondo azul oscuro
4. **En Landing** - CTA verde menta destacado

## PRINCIPIOS DE DISEÑO EXTRAÍDOS

1. ✅ Alto contraste entre fondos oscuros y textos claros
2. ✅ Verde menta como único color "cálido" - muy intencional
3. ✅ Bordes redondeados en todo
4. ✅ Mucho aire blanco/espaciado
5. ✅ Tipografía clara y legible
6. ✅ Escala de grises limitada (solo blanco y negro)
7. ✅ Humanización con fotografías reales
8. ✅ Simplicity and clarity

## RECOMENDACIONES PARA DISEÑO DE INTERFACES

1. **Mantener consistencia de colores** - Usar azul oscuro, verde menta y blanco únicamente
2. **Espaciado generoso** - Mínimo 16px entre bloques
3. **Tipografía clara** - Mont en todos los contextos
4. **Bordes redondeados** - Mínimo 8px en botones, hasta 16px en cards grandes
5. **Alto contraste** - Asegurar legibilidad en todos los estados
6. **Acentos verdes** - Solo para acciones principales (CTA)
7. **Componentes humanizados** - Incluir fotos reales cuando sea posible
8. **Estados visuales** - Hover, active, disabled claramente diferenciados

## RESTRICCIONES DE DISEÑO

❌ NO usar colores saturados o vibrantes
❌ NO usar múltiples acentos de color
❌ NO usar Sans-serif diferentes a Mont
❌ NO usar sombras agresivas
❌ NO usar patrones decorativos complejos
❌ NO abusar de elementos visuales

✅ SÓLO estas dos combinaciones:
- Azul oscuro fondo + blanco texto + verde menta acentos
- Blanco fondo + azul oscuro texto + verde menta acentos
